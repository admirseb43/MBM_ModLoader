# MBM_ModLoader
A mod loader to enhanced your **Monster Black Market (MBM)** (Mature rating game) from [Team-Apple Pie](https://ci-en.dlsite.com/creator/8791) experience. It allows you to manage your mod updates, making it a single click. 



# Country flag
Country flag use came from lipis and his [flag-icons](https://github.com/lipis/flag-icons) repo.
